Introducción

El World Wide Web ha recorrido un largo camino desde sus días como un modesto sistema de hipertexto utilizado por unos pocos científicos para compartir información en Internet. Hoy en día, el World Wide Web es el medio de intercambio de información para millones de personas. Están compartiendo texto, video, sonido y datos, y cada vez más, están tratando de hacer que sus páginas web sean interactivas. Las empresas están tratando de vender sus productos, los artistas están produciendo nuevas formas de arte interactivo, y los programadores están produciendo ayudas para el desarrollo de programas, todo entregado a través del World Wide Web y accesible desde dentro de las aplicaciones de navegador web cotidianas.

En muchos sentidos, estamos en medio de una revolución de la información con un movimiento lejos de la informática centrada en documentos hacia un paradigma centrado en la red. Justo en el centro de este cambio está Netscape Communications y su inmensamente popular navegador web, Netscape Navigator. Al incorporar tecnología de Sun Microsystems y su propio lenguaje de secuencias de comandos JavaScript en la versión actual del navegador Netscape Navigator, Netscape ha ayudado a generar una oleada de movimiento en el World Wide Web dirigida a la creación de documentos e información interactivos.

Java es un lenguaje de programación independiente de la plataforma diseñado para aplicaciones distribuidas en Internet. JavaScript actualmente mejora la funcionalidad del navegador web Netscape Navigator, permitiendo a los autores y desarrolladores web producir contenido que cambia en respuesta a las acciones del usuario: datos dinámicos que hacen que la información sea más accesible y más fácil de organizar y digerir.

En este libro vamos a echar un vistazo a JavaScript, el lenguaje de secuencias de comandos interno que Netscape ha desarrollado e incluido en el navegador Navigator. JavaScript es una herramienta en evolución, al igual que muchas herramientas asociadas con Internet y el World Wide Web. Sin embargo, el futuro de JavaScript es lo suficientemente claro como para que muchas personas ya hayan desarrollado aplicaciones web sofisticadas utilizando el lenguaje.

Objetivo de este Libro

Como mencioné en la Introducción, este libro está diseñado para enseñar el lenguaje de secuencias de comandos JavaScript.

Comenzamos tomando una amplia mirada a Netscape Navigator 3.0, la versión actual del popular navegador web, que según algunas estimaciones controla más del 80 por ciento del mercado de navegadores web. Después de esto, echamos un vistazo introductorio a JavaScript y su relación con Java y su lugar en la suite de herramientas de desarrollo web de Netscape.

Una vez hecho esto, estaremos listos para examinar los entresijos del lenguaje JavaScript y aprender cómo aplicarlos a escenarios del mundo real en la web. JavaScript puede ser utilizado para agregar una amplia gama de interactividad y funcionalidad a páginas web, incluyendo lo siguiente:

Formularios dinámicos que incluyen verificación de errores incorporada

  • Hojas de cálculo y calculadoras
  • Interacción del usuario en forma de mensajes de advertencia y mensajes de confirmación
  • Cambios dinámicos en los colores de texto y fondo
  • La capacidad de analizar URL y acceder a URL en la lista de historial de un usuario
  • La capacidad de abrir, nombrar, limpiar y cerrar nuevas ventanas y dirigir la salida a marcos específicos

Estos tipos de funciones ya aparecen en numerosos sitios web en la World Wide Web, y se espera que el número crezca rápidamente a medida que JavaScript desarrolle madurez como lenguaje de programación.

A lo largo del libro, tendrás la oportunidad de desarrollar varios pequeños scripts que podrás utilizar inmediatamente en tus propias páginas web.

Finalmente, cerramos con una revisión del futuro de JavaScript y hacia dónde parece dirigirse. Consideraremos los planes de Netscape así como los anuncios de otras empresas para incluir JavaScript en sus productos.

¿Quién debería leer este libro?

El lenguaje de secuencias de comandos JavaScript naturalmente interesa a un grupo diverso de personas, incluyendo diseñadores y autores web, programadores y desarrolladores de aplicaciones.

Autores y Diseñadores Web

Este libro es claramente de interés para los desarrolladores y autores web con experiencia en el uso de HTML y en el diseño de sitios web, incluyendo el uso de extensiones de Netscape. Aunque se asume un conocimiento básico de HTML a lo largo del libro, cualquier etiqueta HTML avanzada o complicada que se utilice será introducida y descrita según sea necesario.

Naturalmente, el conocimiento de programación y la comprensión de la Interfaz de Pasarela Común (CGI), utilizada para agregar interactividad en el servidor, es útil en cualquier discusión sobre el desarrollo web. Sin embargo, no es esencial para aprender JavaScript, y los lectores con un conocimiento sólido de las etiquetas HTML pueden seguir las lecciones de este libro. Al final del libro, no solo podrás escribir programas simples (y complejos) en JavaScript, sino que también habrás aprendido a usar algunas de las nuevas extensiones de Netscape para HTML.

Otros

Naturalmente, la discusión sobre un lenguaje de programación como JavaScript no es de interés únicamente para autores y diseñadores web.

Los programadores que buscan añadir la última tecnología a su lista de habilidades están cada vez más interesados en aprender JavaScript. Además, los desarrolladores de aplicaciones que consideran Navigator 2.0 como un motor para implementar aplicaciones de interfaz gráfica de usuario independientes de la plataforma encontrarán rápidamente que JavaScript va a jugar un papel crítico en la implementación de sus aplicaciones.

Preparándose para comenzar

Para aprovechar al máximo este libro, necesitarás varias herramientas. Es esencial tener una copia de la versión más reciente de Netscape 2.0 o 3.0 para desarrollar y probar el código del programa. Además, un buen programa de edición con el que te sientas cómodo facilitará el proceso de desarrollo del programa.

Herramientas de Edición y Desarrollo

Además de una copia de Navigator 3.0, un editor o herramienta de desarrollo sólido facilitará la tarea de escribir, desarrollar y depurar JavaScript.

Si ya haces mucho autoría HTML o programación, probablemente tengas tus propias herramientas favoritas que se adaptarán bien al desarrollo de JavaScript. Siempre y cuando tu software de edición produzca archivos de texto ASCII puro, deberías estar bien.

Sin embargo, varias herramientas pueden facilitar el desarrollo, la edición y la resolución de problemas en tus programas JavaScript.

Al considerar editores, sería útil buscar herramientas que te ayuden a identificar el número de línea actual para depurar scripts. Además, la capacidad de lanzar Netscape Navigator desde un editor es una característica útil que ya se encuentra en muchos editores HTML. La mayoría de los editores HTML son adecuados para el desarrollo de JavaScript, aunque algunos, incluyendo productos líderes como HoT MetaL, están diseñados para realizar validación de HTML y no pueden utilizarse fácilmente para desarrollar scripts JavaScript.
