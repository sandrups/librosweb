Hola Mundo!

Decada de los 90

1991

HTML 1.0

  • Primera versión oficial de HTML, creada por Tim Berners-Lee.
  • Definió la estructura básica de un documento HTML con etiquetas para encabezados, párrafos, listas, etc.

1992

Mosaic

  • El equipo del National Center for Supercomputing Applications (NCSA) en la Universidad de Illinois, liderado por Marc Andreessen, comenzó a desarrollar Mosaic

1993

Mosaic

  • Enero: Se lanza Mosaic 1.0, el primer navegador web gráfico popular que podía mostrar imágenes en línea. Inicialmente solo para Unix X Window System.
  • Octubre: Se lanza Mosaic 2.0, con soporte para Windows.
  • Noviembre: Sale Mosaic 2.1, la primera versión con soporte para Apple Macintosh.

CERN liberó el código fuente de la World Wide Web, permitiendo su uso por cualquier persona.

1994

Larry Aronson. Ziff-Davis Press, Emeryville, California

Mosaic

  • Mosaic 2.4 y 2.5 se lanzan con mejoras incrementales.
  • Marc Andreessen y otros dejan el NCSA para fundar Mosaic Communications Corp (luego Netscape).

PHP 1.0

  • La primera versión de PHP, originalmente llamada "Personal Home Page Tools".
  • Creada por Rasmus Lerdorf para rastrear visitas en su sitio web.

W3C (World Wide Web Consortium)

  • Fue fundada en octubre de 1994.
  • Es un consorcio internacional que produce recomendaciones y estándares para la World Wide Web.
  • Liderada por Tim Berners-Lee, el inventor de la Web.
  • Su objetivo es guiar la evolución de la web hacia su máximo potencial mediante el desarrollo de protocolos y pautas que aseguren su crecimiento a largo plazo.

Netscape

  • Netscape Communications Corporation fue fundada en 1994 por Marc Andreessen y Jim Clark.
  • Utilizaron el código fuente de Mosaic como base para desarrollar un nuevo navegador comercial llamado Netscape Navigator.
  • Netscape Navigator 1.0 fue lanzado en diciembre de 1994 y rápidamente se convirtió en el navegador web líder en el mercado.
  • Netscape fue pionera en la incorporación de nuevas tecnologías web como JavaScript, marcos (frames), cookies y complementos (plugins).
  • A diferencia de Mosaic, que era gratuito, Netscape Navigator fue un producto comercial con características adicionales.

1995

Larry Aronson. Ziff-Davis Press, Emeryville, California

JavaScript 1.0

  • Fue creado por Brendan Eich mientras trabajaba en Netscape, inicialmente se llamaba Mocha, luego se renombró a LiveScript y finalmente a JavaScript.
  • Lanzado en diciembre de 1995 con Netscape Navigator 2.0.
  • JavaScript fue diseñado para ser un lenguaje de scripting ligero, interpretado y orientado a objetos, adecuado para agregar interactividad y comportamiento dinámico a las páginas web.

MySQL 1.0

  • La primera versión de MySQL es publicada por la empresa MySQL AB.

PHP 2.0

  • Introdujo la integración con bases de datos y una mejor arquitectura modular.

HTML 2.0

  • Introdujo características adicionales como formularios y tablas.
  • Estandarizado por el IETF (Internet Engineering Task Force).

Internet Explorer 1.0

  • Microsoft lanza Internet Explorer 1.0, una versión básica basada en Mosaic.

1996

Arman Danesh | Sams.net. Indianapolis, Indiana (noviembre 1996)

JavaScript 1.1

  • Fue lanzado en agosto de 1996 por Netscape Communications Corporation. Esta versión se incluyó en Netscape Navigator 3.0
  • Introdujo la captura de eventos, la manipulación del DOM y algunas características adicionales.

CSS 1

  • 17 de diciembre de 1996: La primera especificación oficial de CSS, publicada por el W3C.
  • Definió la forma básica de asociar estilos con elementos HTML.

MySQL 3.11

  • Introducción de funciones importantes como el manejo de contraseñas y registros de consultas.

JScript 1.0

  • JScript fue introducido por Microsoft en agosto de 1996 junto con el lanzamiento de Internet Explorer 3.0.
  • Se basó en gran medida en la especificación inicial de JavaScript desarrollada por Netscape

Internet Explorer 3.0

  • Incorporación de soporte para hojas de estilo CSS, transferencia de archivos vía FTP, firma de código y otras mejoras.

PostgreSQL

  • Fue desarrollado a partir del proyecto POSTGRES en la Universidad de California, Berkeley, liderado por Michael Stonebraker

1997

Larry Aronson y Joseph Lowery. Ziff-Davis Press. Emeryville, California

Hakon Wium Lie, Bert Bos | Addison Wesley Longman. Harlow, England

HTML 3.2

  • Fue lanzado en enero de 1997 por el World Wide Web Consortium (W3C)
  • Esta versión estandarizó y amplió muchas características que se habían desarrollado de manera independiente en diferentes navegadores.

JavaScript 1.2

  • Introducido en Netscape Navigator 4.0 en junio de 1997
  • Mejor integración con CSS, permitiendo una manipulación más dinámica del estilo de las páginas web.
  • Se añadió soporte para expresiones regulares, lo que permitió una manipulación y validación de cadenas más potente y flexible.

ECMAScript 1.0

  • Se llamó oficialmente ECMA-262 1ª Edición, fue publicada en junio de 1997.
  • Se basó principalmente en JavaScript 1.1

HTML 4.0

  • Fue lanzado en Diciembre de 1997
  • Se lanzó en tres variantes: Strict, Transitional, y Frameset

Internet Explorer 4.0

  • Llega Internet Explorer 4.0 con una nueva interfaz, favoritos, barra de herramientas personalizable y otras mejoras.

1998

XML

  • (eXtensible Markup Language) fue lanzado en Febrero de 1998
  • Diseñado para almacenar y transportar datos de forma legible tanto para humanos como para máquinas.

CSS2

  • (Cascading Style Sheets, Level 2) fue publicado por el World Wide Web Consortium (W3C) como una recomendación oficial el 12 de mayo de 1998.

JavaScript 1.3

  • Fue lanzado en octubre de 1998, y se incluyó en Netscape Navigator 4.06
  • Se alineaba en gran medida con la especificación ECMAScript 1, que se había publicado en junio de 1997.
  • Características nuevas: Introducción de la declaración 'let' para variables de bloque, Adición del método 'toSource()' para objetos, Implementación del operador 'in' para verificar si una propiedad existe en un objeto.

1999

objeto XMLHttpRequest

  • Fue introducido en marzo de 1999, por Microsoft en Internet Explorer 5 como un control ActiveX, bajo el nombre de XMLHTTP
  • Este objeto permitió a los desarrolladores enviar y recibir datos de un servidor de manera asíncrona sin necesidad de recargar la página completa.
  • El objeto XMLHttpRequest en su forma original fue parte del lenguaje de scripting de Microsoft conocido como JScript.

JavaScript 1.4

  • Lanzado en noviembre de 1999, principalmente utilizado en el servidor con Netscape Enterprise Server. Fue lanzado junto con Netscape Navigator 4.7
  • Nuevos métodos de Array: Se añadieron métodos como concat(), slice(), y splice().

HTML 4.01

  • lanzado en diciembre de 1999
  • Se mejoró la compatibilidad con XHTML 1.0, que se estaba desarrollando en paralelo.
  • Se añadieron algunos atributos nuevos a elementos existentes para mejorar la funcionalidad.
